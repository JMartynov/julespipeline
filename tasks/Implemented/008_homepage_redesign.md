# Task 8: Homepage Redesign

## Subtasks

### 8.1: Hero Section
- [ ] Design modern hero with animated background
- [ ] Headline: Clear value proposition (40-50 chars)
- [ ] Subheading: Description (80-100 chars)
- [ ] CTA button: "Get Started Free"
- [ ] Background animation (gradient shift, particles, or subtle pattern)
- [ ] Responsive: Mobile/tablet/desktop layouts

### 8.2: Tool Showcase Grid
- [ ] 5-tool grid layout (responsive: 1 col mobile, 2-3 cols tablet, 5 cols desktop)
- [ ] Per tool card:
  - [ ] Icon/illustration (consistent style)
  - [ ] Tool name (heading)
  - [ ] Brief description (2-3 lines)
  - [ ] Quick stat (e.g., "Extract 10 tables/min")
  - [ ] "Try it free" CTA button
  - [ ] Hover effects (shadow lift, color shift)

### 8.3: Social Proof Section
- [ ] Testimonials carousel (3-5 real or sample quotes)
- [ ] Usage stats (e.g., "10K+ PDFs processed monthly")
- [ ] Trust badges/certifications (if applicable)
- [ ] Responsive layout

### 8.4: FAQ Accordion
- [ ] 5-7 common questions
- [ ] Smooth open/close animation
- [ ] Clear answer text
- [ ] Keyboard accessible

### 8.5: Footer
- [ ] Links: About, Blog, Pricing, Docs
- [ ] Newsletter signup form
- [ ] Social media icons (if applicable)
- [ ] Copyright, Privacy Policy, Terms
- [ ] Responsive layout

### 8.6: Animations & Interactions
- [ ] Smooth scroll behavior
- [ ] Fade-in on scroll (for elements)
- [ ] Button hover state (color shift, shadow)
- [ ] Card hover (lift effect)
- [ ] Icon animations (on hover or scroll)

### 8.7: Dark Mode Support
- [ ] All colors have dark variants
- [ ] Gradient backgrounds adapt
- [ ] Text contrast in both modes
- [ ] Smooth toggle switch

### 8.8: Responsive Design
- [ ] Mobile-first approach
- [ ] Breakpoints: 375px, 768px, 1024px, 1440px
- [ ] Touch-friendly buttons (44x44px min)
- [ ] Single column on mobile
- [ ] Test on real devices

### 8.9: SEO Optimization
- [ ] Semantic HTML (h1, h2, nav, etc.)
- [ ] Meta tags (title, description)
- [ ] Open Graph tags (social sharing)
- [ ] Structured data (Schema.org)

### 8.10: Performance
- [ ] Optimize images (WebP with PNG fallback)
- [ ] Lazy load below-fold images
- [ ] Minimize CSS/JS
- [ ] Lighthouse score 90+

- Image assets (optimized)
