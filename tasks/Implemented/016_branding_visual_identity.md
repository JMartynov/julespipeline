# Task 16: Branding & Visual Identity

## Subtasks

### 16.1: Logo Guidelines
- [ ] Logo placement rules (header, footer, etc.)
- [ ] Sizing guidelines (min 40px width)
- [ ] Spacing around logo (clear space)
- [ ] Logo in light and dark modes
- [ ] Favicon setup (16x16, 32x32, SVG)
- [ ] Logo file formats (SVG, PNG)

### 16.2: Hero Image/Illustration Library
- [ ] Tool-specific illustrations (5 total, one per tool)
- [ ] Processing state graphics (loading, success, error)
- [ ] Empty state illustrations (no data, no results)
- [ ] Consistent style (line weight, colors, perspective)
- [ ] Responsive sizes (mobile, tablet, desktop)

### 16.3: Brand Colors Applied
- [ ] Primary color: consistent across site
- [ ] Secondary colors: for accents
- [ ] Status colors: green (success), red (error), yellow (warning)
- [ ] Neutral palette: grays for text/backgrounds
- [ ] All colors documented in brand guidelines

### 16.4: Typography Hierarchy
- [ ] H1: 40px, bold, primary color
- [ ] H2: 32px, bold, primary/secondary
- [ ] H3: 24px, semibold
- [ ] Body: 16px, regular, neutral gray
- [ ] Small: 12px, regular, light gray
- [ ] Clear hierarchy throughout site

### 16.5: Consistent Spacing & Alignment
- [ ] Grid system applied consistently
- [ ] Margins/padding follow spacing scale
- [ ] All elements aligned to grid
- [ ] No random spacing
- [ ] Consistent padding in cards/containers

### 16.6: Professional Gradients (If Applicable)
- [ ] Hero gradient (if used)
- [ ] Button gradients (subtle)
- [ ] Background gradients (very subtle, not distracting)
- [ ] Consistent gradient direction
- [ ] 2-3 gradient palettes max

### 16.7: Illustrations & Icons
- [ ] Icons reflect brand personality (modern, clean, approachable)
- [ ] Consistent stroke width (2px)
- [ ] Consistent sizing (24x24, 32x32)
- [ ] Consistent fill/stroke approach
- [ ] Avoid generic icon packs (customize if needed)

### 16.8: Photography Style
- [ ] If using photography: bright, modern, people-focused
- [ ] Consistent color grading
- [ ] Consistent composition style
- [ ] High quality (no pixelated images)
- [ ] Optimized for web (compressed)

### 16.9: Brand Personality
- [ ] Tone of voice guidelines (friendly, professional, helpful)
- [ ] Language style (avoid jargon, be clear)
- [ ] Visual style (modern, not outdated)
- [ ] Accessibility is part of brand (inclusive design)

### 16.10: Create Brand Guidelines Document
- [ ] Brand story / values
- [ ] Logo guidelines
- [ ] Color palette (with hex codes)
- [ ] Typography rules
- [ ] Imagery style
- [ ] Tone of voice
- [ ] Dos and don'ts
- [ ] File: `docs/brand/BRAND_GUIDELINES.md`

### 16.11: Create Icon/Illustration Library
- [ ] Figma file with all icons (or web-based)
- [ ] All illustrations (tool-specific, states, empty states)
- [ ] Export for web (SVG)
- [ ] Organized in folders (tools, states, misc)

### 16.12: Visual Consistency Check
- [ ] All pages use same typography
- [ ] All pages use same colors
- [ ] All buttons/cards look consistent
- [ ] No mismatched styles
- [ ] Create visual design QA checklist

- Icon/illustration library (Figma or similar)
- Logo files (SVG, PNG)
- Color palette reference
- Typography scales
