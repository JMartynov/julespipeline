# Task: 02 String Parsing

Add a feature to parse simple mathematical expressions from strings (e.g., '2 + 3 * 4') and evaluate them following standard order of operations. 

Requirements:
1. Implement a parser that handles addition, subtraction, multiplication, and division.
2. Respect operator precedence (multiplication and division before addition and subtraction).
3. Support parentheses for grouping.
4. Integrate this with the existing `math_operations.py` logic.
5. Add comprehensive unit tests in `test_string_parsing.py`.
